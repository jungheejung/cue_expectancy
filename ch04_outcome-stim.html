Chapter 5 [ beh ] outcome ~ stimulus_intensity

What is the purpose of this notebook?

Here, I plot the outcome ratings as a function of stimulus intensity * Main model: lmer(outcome_rating ~ stim) * Main question: do outcome ratings differ as a function of stimulus intensity? We should expect to see a linear effect of stimulus intensity. * If there is a main effect of cue on expectation ratings, does this cue effect differ depending on task type? * IV: stim (high / med / low) * DV: outcome rating
